Sandwich.Me

Fall 2009

Sandwich.Me is part of a project to develop an interactive Web 2.0 application with business potential. It's a recipe site specifically for delicious sandwiches, and uses IP geolocation to find the nearest store from which to buy materials. While an actual implementation of the application was not necessary, I went ahead and made a full-scale mock-up for our presentation.
